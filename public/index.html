For aggregators & operators. This is a demonstration build in play-money mode — balances reset, no real funds. Outcomes, balances and round timing are decided server-side; the browser is a thin renderer. Every round is provably fair: the server seed is committed before play (a pre-published hash chain) and revealed for verification. The integration surface is a single WebSocket + an idempotent wallet (debit / credit / rollback by txId) — one connection serves the whole catalogue.
